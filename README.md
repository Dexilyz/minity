# Minity

Визуальный конструктор действий для Minecraft (Fabric, 1.26.1) в стиле Scratch.
Собирай из блоков, что делает мод: события → действия, с условиями `если`, циклами и темами.

## Установка
Скачай `minity-<версия>.jar` и положи в папку `mods`. Нужен Fabric API.

## Обновления
Мод сам проверяет новые версии (файл `version.json` в этом репозитории) и предлагает обновиться в игре.
